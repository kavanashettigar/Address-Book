# Address-Book
This project is a digital telephone directory that allows users to store and manage contact information such as name, mobile number, and email ID.

Features
- Add new contact
- Search contact by name or number
 -Edit existing contact details
 -Delete contact
 -Display all contacts
 -File storage for permanent data saving

Technologies Used
 -C Programming (Advanced C Concepts)
 -File Handling
 -Structures
 -Pointers
 -Functions
 -Makefile

 How to Run
 1.Compile the program using: gcc *.c -o addressbook
 2.Run the executable using: ./addressbook
